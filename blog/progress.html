环形进度条的两种实现

业务开发中经常有用到环形进度条的场景，环形进度条可以用 Canvas 和 Svg 来实现，接下来分别尝试下。

# Canvas 实现

环形进度条是由两个圆叠加而成的，背景圆和当前进度圆环，Canvas 画圆用 arc 方法，可以创建圆和部分圆，arc 方法用法如下。

context.arc(x, y, r, sAngle, eAngle, counterclockwise);

# arc 参数说明

参数 说明
x 圆的中心的 x 坐标
y 圆的中心的 y 坐标
r 圆的半径
sAngle 起始角，以弧度计（弧的圆形的三点钟位置是 0 度）
eAngle 结束角，以弧度计
counterclockwise 可选。规定应该逆时针还是顺时针绘图。False = 顺时针，true = 逆时针。

# 背景圆环

来画一个背景圆环

<template>
  <canvas class="canvas" ref="canvas" width="140" height="140"></canvas>
</template>

<script>
export default {
  mounted() {
    this.drawCircle()
  },

  methods: {
    drawCircle() {
      const canvas = this.$refs.canvas
      const ctx = canvas.getContext('2d')

      // 绘制背景圆环
      ctx.beginPath()
      ctx.lineWidth = 8
      ctx.strokeStyle = '#EEEEEE'
      ctx.arc(70, 70, 62, 0, 2 * Math.PI, false)
      ctx.stroke()
    }
  }
}
</script>

# 进度圆环

接着在背景圆环上绘制蓝色进度圆环，进度为25%，也就是 1/4 圆，默认情况下绘制圆的起点是 3 点钟方向，但设计师一般要求圆的起点是 12 点方向，所以绘制进度圆环的时候要加 1.5π，这样就从 3 点钟方向按顺时针跑到 12 点钟方向了，那么结束点就是 1.5π + 0.5π 了，为什么是 0.5π 呢，因为一个整圆是 2π，那么 1/4 圆就是 2π * 1/4 = 0.5π 了。























 
 
 
 
 
 





<template>
  <canvas class="canvas" ref="canvas" width="140" height="140"></canvas>
</template>

<script>
export default {
  mounted() {
    this.drawCircle()
  },

  methods: {
    drawCircle() {
      const canvas = this.$refs.canvas
      const ctx = canvas.getContext('2d')

      // 绘制背景圆环
      ctx.beginPath()
      ctx.lineWidth = 8
      ctx.strokeStyle = '#EEEEEE'
      ctx.arc(70, 70, 62, 0, 2 * Math.PI, false)
      ctx.stroke()

      // 绘制进度圆环
      ctx.beginPath()
      ctx.lineWidth = 8
      ctx.strokeStyle = '#466796'
      ctx.arc(70, 70, 62, 1.5 * Math.PI, 2 * Math.PI, false)
      ctx.stroke()
    }
  }
}
</script>

# 提取 props

接下来把大小、进度、颜色、线条粗细等提炼成 props，并设置一些默认值







 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 
 

 
 
 
 
 






























<template>
  <canvas class="canvas" ref="canvas" :width="width" :height="width"></canvas>
</template>

<script>
export default {
  props: {
    // 进度
    percentage: {
      type: Number,
      default: 0
    },
    // 画布大小
    width: {
      type: Number,
      default: 140
    },
    // 背景圆环颜色
    backgroundColor: {
      type: String,
      default: '#EEEEEE'
    },
    // 圆环宽度
    strokeWidth: {
      type: Number,
      default: 8
    },
    // 进度圆环颜色
    strokeColor: {
      type: String,
      default: '#466796'
    },
  },

  computed: {
    endAngle () {
      return (parseFloat(this.percentage / 100) * 2 + 1.5) * Math.PI
    }
  },

  mounted() {
    this.drawCircle()
  },

  methods: {
    drawCircle() {
      const canvas = this.$refs.canvas
      const ctx = canvas.getContext('2d')
      const center = this.width / 2 // 圆环中心点坐标
      const radius = center - this.strokeWidth / 2  // 半径

      // 绘制背景圆环
      ctx.beginPath()
      ctx.lineWidth = this.strokeWidth
      ctx.strokeStyle = this.backgroundColor
      ctx.arc(center, center, radius, 0, 2 * Math.PI, false)
      ctx.stroke()

      // 绘制进度圆环
      ctx.beginPath()
      ctx.lineWidth = this.strokeWidth
      ctx.strokeStyle = this.strokeColor
      ctx.arc(center, center, radius, 1.5 * Math.PI, this.endAngle, false)
      ctx.stroke()
    }
  }
}
</script>

# slot 插槽

有的时候需要在环形进度条内部展示一些自定义内容，比如进度或者图标等，所以提供一个 slot 插槽。

<!-- progress.vue -->
<template>
  <div class="progress" :style="{ width: width + 'px', height: width + 'px' }">
    <canvas class="canvas" ref="canvas" :width="width" :height="width"></canvas>
    <div class="progress-slot">
      <slot></slot>
    </div>
  </div>
</template>

<!-- 调用 -->
<Progress :percentage="75">
  <p class="percentage">75<span class="symbol">%</span></p>
</Progress>

# 动画效果

到目前为止环形进度条已经很完善了，但仔细看进度条在初始化渲染和动态修改的时候没有动画效果，我们来加下动画效果。Canvas 动画只能用笨方法，既一帧一帧的重绘，每隔几毫米更新一次进度，然后清空画布重绘。

<script>
export default {
  data() {
    return {
      endAngle: 0
    }
  },

  watch: {
    percentage(newValue, oldValue) {
      this.drawProgress(oldValue)
    }
  },

  mounted() {
    this.drawProgress()
  },

  methods: {
    async drawProgress(start = 0) {
      // 动态绘制进度条，一秒内绘制完成，绘制间隔16.7毫秒
      const step = this.percentage > start ? 1 : -1
      let percentage = start
      let i = 0
      while(percentage !== this.percentage) {
        percentage = start + step * i
        this.endAngle = (percentage / 100 * 2 + 1.5) * Math.PI
        await this.sleep(1000/60)
        this.drawCircle()
        i++
      }
    },

    // 等待多少毫秒
    sleep(ms) {
      return new Promise(resolve => setTimeout(resolve, ms))
    }
  }
}
</script>

# SVG 实现

虽然 Canvas 实现起来比较简单粗暴，但是 Canvas 会有毛刺，有理想的设计师肯定不同意的，那我们来试试 SVG 方案。SVG 画圆也很简单，用 circle 标签画两个圆，先画背景圆环，再画进度圆环。SVG circle 没有提供画部分圆的方法，只能通过属性 stroke-dasharray 来实现，它是一个 length 和 percentage 数列，数与数之间用逗号或者空白隔开，指定短划线和缺口的长度。比如我们设置 stroke-dasharray="10, 20"，就会先画10px的短划线，然后空20px缺口，一直循环下去，如下所示。

此时我们想要绘制进度条，那是否可以变换下思路，比如进度是 50%，那么设置短划线为圆周长的一半，然后缺口是整个周长，来试试看效果，还不错，就是起点在 3 点钟方向，要改到 12 点钟方向，SVG 可以通过 CSS transform: rotate(-90deg) 逆时针旋转 90° 设置。

<!-- progress.vue -->
<template>
  <div :style="{ width: width + 'px', height: width + 'px' }" class="progress-svg">
    <svg :viewBox="`0, 0, ${width}, ${width}`">
      <!-- 背景圆环 -->
      <circle
        :cx="width / 2"
        :cy="width / 2"
        :r="radius"
        :stroke-width="strokeWidth"
        :stroke="backgroundColor"
        fill="none"
        stroke-linecap="butt"
      ></circle>
      <!-- 进度条圆环 -->
      <circle
        class="progress-circle"
        :cx="width / 2"
        :cy="width / 2"
        :r="radius"
        :stroke-width="strokeWidth"
        :stroke="strokeColor"
        fill="none"
        stroke-linecap="butt"
        :stroke-dasharray="`${strokeLength}, ${perimeter}`"
      ></circle>
    </svg>

    <div class="progress-slot">
      <slot></slot>
    </div>
  </div>
</template>

<script>
export default {
  props: {
    percentage: {
      type: Number,
      default: 0
    },
    width: {
      type: Number,
      default: 140
    },
    backgroundColor: {
      type: String,
      default: '#EEEEEE'
    },
    strokeWidth: {
      type: Number,
      default: 8
    },
    strokeColor: {
      type: String,
      default: '#466796'
    },
  },

  computed: {
     // 半径
    radius() {
      return (this.width - this.strokeWidth) / 2 
    },

    // 周长, 2πR
    perimeter() {
      return 2 * Math.PI * this.radius
    },

    // 进度条绘制长度，周长 * 当前进度百分比
    strokeLength() {
      return this.percentage / 100 * this.perimeter
    }
  }
}
</script>

<style lang="scss" scoped>
.progress-circle {
  transform: rotate(-90deg);
  transform-origin: center;
  transition: all 0.3s;
}

.progress-svg {
  position: relative;
}

.progress-slot {
  position: absolute;
  top: 0;
  left: 0;
  right: 0;
  bottom: 0;
  display: flex;
  align-items: center;
  justify-content: center;
}
</style>
